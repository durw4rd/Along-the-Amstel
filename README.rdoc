== README

Project started in TDD environment, exploring the possibilities of Sass tools.

<h1>Stack</h1>

The application was built using Ruby on Rails; Sass tool used: Bourbon (incl. Neat & Refills)

<h1>Running the App</h1>

DB initialization:

<tt>rake db:create db:migrate</tt>

Bundling

<tt>bundle install</tt>

Start server

<tt>rails server</tt>
